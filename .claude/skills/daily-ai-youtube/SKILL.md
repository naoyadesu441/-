# daily-ai-youtube — 海外AIニュース YouTube動画台本 自動生成

Stage 1（GitHub Actions）が毎朝 `news/YYYY-MM-DD.md` に「日本語要約＋裏取りステータス＋元リンク」を出力する。このスキルはそれを入力に、なおや式の動画台本を作って `scripts/youtube/YYYY-MM-DD.md` に書き出し、コミットする。

## 手順

1. **当日のリサーチを読む**
   - リポジトリ直下の `news/` から当日（JST）の `YYYY-MM-DD.md` を読む。
   - 無ければ `news/` 内の最新ファイルを使い、「最新分で代用した」旨を後述の出力末尾に注記する。

2. **台本にするネタを選ぶ（2〜4本）**
   - **裏取りステータスを最優先**で判断する:
     - 🟢一次確認済 / 🟡二次 を優先採用。
     - 🔴未確認は原則除外。どうしても扱う価値がある場合のみ採用し、台本中に必ず「※これは未確認情報です」と明示する。
   - 視聴者（AIを仕事で使いたい一般〜中級者）にとっての「実用性・驚き・話題性」で2〜4本に絞る。
   - 同じ話題の重複は1本にまとめる。

3. **台本を書く**
   - `台本原則.md`（同フォルダ）を必ず読み、その構成・トーン・尺・禁止事項に従う。
   - 各ネタを **共感フック → 具体数字 → 解決プロセス → 「実はこんなに簡単」 → CTA** の型で日本語の話し言葉にする。
   - 台本の各セクション末尾に、根拠として「元リンク」と「裏取りステータス」を残す（撮影者が確認できるように）。

4. **書き出す**
   - `scripts/youtube/YYYY-MM-DD.md`（JSTの当日日付）に保存する。
   - フォーマットは `台本原則.md` の「出力テンプレート」に従う。

5. **コミット & プッシュ**
   - `git add scripts/youtube/` → `git commit -m "youtube: YYYY-MM-DD AIニュース動画台本"` → `git push`。
   - ブランチは Routine の設定に従う（通常 main、またはレビュー運用なら作業ブランチ＋PR）。

6. **（任意）通知**
   - 必要に応じて Notion（台本DBがあれば）や Discord に、Claudeセッション内の MCP（Notion / GitHub）で完了を通知する。指示が無ければ通知は省略してよい。

## 注意
- リサーチに無い事実を創作しない。数字・固有名詞はリサーチ（および元リンク）の範囲に収める。
- 裏取りできていない内容を断定口調で語らない。
- 出力は日本語。なおやのチャンネルのトーンに合わせる（`台本原則.md` 参照）。
